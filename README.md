# RAG-project

Интерактивное Streamlit-приложение для исправления фактических ошибок и опечаток с помощью Retrieval-Augmented Generation (RAG).
Система автоматически исправляет текст, опираясь на данные из Википедии и собственного векторного хранилища на базе Chroma, которое обеспечивает быстрый поиск релевантной информации.

Исправление фактических ошибок
Система анализирует каждое предложение, находит релевантные документы в Chroma-векторной базе и формирует корректное утверждение на основе данных из Wikipedia.

Исправление орфографии
Перед проверкой фактической информации текст проходит автоматическую орфографическую корректировку с помощью YandexSpeller.

## Запуск приложения

```
git clone git@github.com:AleksandrAstakhov/RAG-project.git
```

```
cd RAG-project
pip install -r requirements.txt
```

из корневой паки
```
streamlit run rag/ui.py
```
